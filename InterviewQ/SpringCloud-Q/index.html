# 请说一下你对分布式和微服务的理解

分布式系统：多台计算机协同工作的系统，通过网络通信共同完成任务，对用户表现为一个统一的整体。核心解决扩展性和可靠性问题，需处理网络延迟、数据一致性等挑战。

微服务：一种细粒度的分布式架构风格，将应用拆分为小型、独立的服务（每个服务专注单一业务能力），强调业务解耦
