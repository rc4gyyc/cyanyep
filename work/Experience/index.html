# 白嫖网站付费内容

# 有免费访问次数

如果网站提供了免费访问次数，特别是不用登录的那种，大概率是在前端计算的，

我们可以在localStore中查看有没有关于访问次数的单词，或者每次访问时都会变化的数值，

然后就可以用ai写一个油猴脚本，自动执行修改访问次数，实现付费内容白嫖。
