# 遇到的挑战

在搭建博客网站时，由于参考大佬{ 从零开始搭建博客系列 | 从 01 开始 (opens new window)、VuePress 博客搭建系列 33 篇正式完结！ (opens new window)}的搭建过程，所以大部分都是可以参考着搭建，但是由于选择不同（主打一个零成本(●'◡'●)，不过如果想要看起来更高级，使用更方便，还是需要一些成本），所以肯定无法全部照搬。还是会遇到一些困难与挑战的。

# 最后更新时间（LastUpdated）

一开始所有文章的更新时间总是一样，搞了一天都搞不定

最后还是了解了一下Lastupdated的设置原理，知道是根据git log设置的时间后，问AI怎么在GitHub Action中获取到.git仓库才解决的，

所以要对某事下手，还是得先了解其原理，对其一知半解，也会难以下手。

# 图床

一开始选择七牛云图床，以为可以免费使用，结果免费域名只有三个月的，还是http的域名，

最后找到在阿里云买了个域名，用阿里云的免费证书才能够长期使用，不过七牛云的https不是免费的，CDN流量0.28/GB，

主打零成本当然要换一个啦(●'◡'●)

最后选择了GitHub作为图床，加上使用第三方CDN加速，就是上传图片不一定能成功

# 同步仓库

在同步gitee仓库时，想要部署Gitee Pages，总是报错，最后想要先直接在gitee上试着Pages一下发现gitee Pages服务已暂停/(ㄒoㄒ)/~~
