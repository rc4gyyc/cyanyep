通过自定义脚本+自定义标记，集合titleTag、article、hide标记，实现文章的隐藏

2025-04-5 13:13:22 @Cyan

  1. 添加标记 在需要隐藏的文章的frontmatter中添加自定义标记，例如：
---
hide: true
---
  1. 修改自动侧边栏配置 在.vuepress/config.js中配置@vuepress/plugin-auto-sidebar插件：
module.exports = {
  plugins: [
    [
      '@vuepress/plugin-auto-sidebar',
      {
        ignore: (filename) => {
          const content = fs.readFileSync(filename, 'utf8');
          return content.match(/^hide:\s*true/m);
        }
      }
    ]
  ]
}
  1. 处理文件名序号 可以添加一个构建脚本（如preprocess.js）：
const fs = require('fs');
const path = require('path');

const docsDir = path.join(__dirname, 'docs');
const files = fs.readdirSync(docsDir);

files.forEach(file => {
  const filePath = path.join(docsDir, file);
  const content = fs.readFileSync(filePath, 'utf8');
  
  if (content.match(/^hide:\s*true/m)) {
    // 去掉序号
    const newName = file.replace(/^\d+\./, '');
    fs.renameSync(filePath, path.join(docsDir, newName));
  }
});
  1. **在package.json中添加预处理命令
{
  "scripts": {
    "predev": "node preprocess.js",
    "dev": "vuepress dev docs"
  }
}

这种方案可以实现：

  • 通过hide: true标记识别需要隐藏的文章
  • 自动去掉文件名中的序号
  • 不会出现在自动生成的侧边栏中
  • 仍可通过URL直接访问
  1. 通过 Frontmatter 自定义标记 在 Markdown 文件头部添加自定义字段（如 hide: true）：
---
hide: true  # 自定义标记
permalink: /custom-url/  # 保留固定URL
---

然后通过插件或主题配置读取该标记

  1. 通过插件处理自定义标记 在 .vuepress/config.js 中配置插件识别标记：

js

复制

module.exports = {
  plugins: [
    {
      name: 'custom-mark-plugin',
      extendsPageOptions: (page) => {
        if (page.frontmatter.hide) {
          return { deactivate: true }  // 隐藏页面
        }
      }
    }
  ]
}
  1. 通过文件名规则

  • 使用 _ 开头的文件/目录会被自动忽略（如 _hidden.md）

  • 去掉序号前缀（如将

    1.article.md

    改为

    article.md

    ）

  1. 通过增强脚本 在 .vuepress/enhanceApp.js 中处理自定义逻辑：
export default ({ Vue }) => {
  Vue.mixin({
    computed: {
      $hiddenPages() {
        return this.$site.pages.filter(p => p.frontmatter.hide)
      }
    }
  })
}
